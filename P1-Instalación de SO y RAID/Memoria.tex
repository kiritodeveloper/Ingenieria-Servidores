	\subsection{Concepto de Máquina Virtual y virtualización}
		\item ¿Qué modos y/o tipos de $"$Virtualización Hardware$"$ existen?(No más de 3 párrafos)
		La virtualización hardware enmascara las características físicas del
		ordenador y las simula de una forma más abstracta. Se pueden dividir en:
		
		\begin{itemize}
			\item \textbf{Virtualización completa}\cite{Full_Virt}\cite{HW_Virt}: en la cuál,
			se muestran suficientes características hardware como para simular la ejecución
			de un sistema operativo de forma que parezca que $"$se está ejecutando en una máquina
			aislada$"$\cite{Intel_Virt}, es decir, como si se ejecutara en un ordenador
			independiente con las características que está proporcionando la VMSW.
			
			\item \textbf{Virtualización parcial}\cite{HW_Virt}: en la cuál, se muestran
			suficientes características como para ejecutar una serie de aplicaciones pero no
			como para ejecutar un sistema operativo.
		\end{itemize}
